
-Instalación

git clone https://github.com/GuadalupeSoria/Collectibol.git

cd Collectibol

npm install --legacy-peer-deps

npm start

Escanea el código QR con Expo Go en el teléfono para ejecutar la app.

-Cómo funciona

Cada vez que disparas la pelota al arco, se genera una letra 3D

Cuando recolectas una letra aparece una carta 3D que puedes rotar 360 grados arrastrando con el dedo y voltear tocándola.

Al completar las 11 letras que formar la palabra "Collectibol" aparece un modal de victoria.
